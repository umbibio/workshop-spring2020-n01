## Install git

### Windows

Choose one of these:

- https://gitforwindows.org/ (preferred)
- https://desktop.github.com/ 

### MacOS

- use homebrew (preferred):
    - https://brew.sh/
    - ` $ brew install git`
- https://desktop.github.com/

### Linux

Use the corresponding package manager:

- debian/ubuntu
    - ` $ sudo apt install git`
- rh/fedora
    - ` $ sudo yum install git`
